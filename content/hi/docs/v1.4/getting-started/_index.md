यह ट्यूटोरियल आपको Cozystack क्लस्टर की आपकी पहली तैनाती से होकर मार्गदर्शन करेगा।
इस दौरान आप प्रमुख अवधारणाओं से परिचित होंगे, डैशबोर्ड और CLI के माध्यम से Cozystack का उपयोग करना सीखेंगे, और एक कार्यशील proof-of-concept प्राप्त करेंगे।

ट्यूटोरियल को कई चरणों में बाँटा गया है। अगला चरण शुरू करने से पहले प्रत्येक चरण पूरा करना सुनिश्चित करें:

| चरण                                                                               | विवरण                                                                                                                          |
|-----------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| [आवश्यकताएँ: इन्फ्रास्ट्रक्चर और टूल तैयार करें]({{% ref "requirements" %}})        | यह ट्यूटोरियल चलाने से पहले इन्फ्रास्ट्रक्चर तैयार करें और अपनी मशीन पर आवश्यक CLI टूल इंस्टॉल करें।                             |
| 1. [Talos Linux इंस्टॉल करें]({{% ref "install-talos" %}})                        | [`boot-to-talos`][btt] का उपयोग करके Talos Linux का Cozystack-विशिष्ट वितरण इंस्टॉल करें — संभवतः सबसे आसान तरीका।             |
| 2. [Kubernetes क्लस्टर इंस्टॉल और बूटस्ट्रैप करें]({{% ref "install-kubernetes" %}}) | Cozystack के लिए बनाए गए Talos कॉन्फ़िगरेशन प्रबंधन टूल [Talm][talm] का उपयोग करके Kubernetes क्लस्टर बूटस्ट्रैप करें।          |
| 3. [Cozystack इंस्टॉल और कॉन्फ़िगर करें]({{% ref "install-cozystack" %}})          | Cozystack इंस्टॉल करें, प्रशासनिक एक्सेस प्राप्त करें, बुनियादी कॉन्फ़िगरेशन करें और Cozystack डैशबोर्ड एक्सेस करें।            |
| 4. [उपयोगकर्ताओं और टीमों के लिए टेनेंट बनाएँ]({{% ref "create-tenant" %}})         | एक उपयोगकर्ता टेनेंट बनाएँ — Cozystack में RBAC की नींव — और डैशबोर्ड तथा Cozystack API के माध्यम से उस तक पहुँचें।            |
| 5. [प्रबंधित एप्लिकेशन परिनियोजित करें]({{% ref "deploy-app" %}})                  | Cozystack का उपयोग शुरू करें: एक वर्चुअल मशीन, प्रबंधित एप्लिकेशन और एक टेनेंट Kubernetes क्लस्टर परिनियोजित करें।             |

[btt]: https://github.com/cozystack/boot-to-talos
[talm]: https://github.com/cozystack/talm
