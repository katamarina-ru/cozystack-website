यह ट्यूटोरियल आपको Cozystack क्लस्टर के पहले परिनियोजन में मार्गदर्शन करेगा।
इस दौरान आप प्रमुख अवधारणाओं से परिचित होंगे, डैशबोर्ड और CLI के माध्यम से Cozystack का उपयोग करना सीखेंगे,
और एक कार्यशील PoC प्राप्त करेंगे।

ट्यूटोरियल को कई चरणों में बाँटा गया है।
अगला चरण शुरू करने से पहले प्रत्येक चरण पूरा करना सुनिश्चित करें:

| चरण                                                                               | विवरण                                                                                                                          |
|-----------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| [आवश्यकताएँ: इन्फ्रास्ट्रक्चर और टूल तैयार करें](/docs/v1.4/getting-started/requirements/)        | इस ट्यूटोरियल को शुरू करने से पहले इन्फ्रास्ट्रक्चर तैयार करें और अपनी मशीन पर आवश्यक CLI टूल इंस्टॉल करें।                     |
| 1. [Talos Linux इंस्टॉल करें](/docs/v1.4/getting-started/install-talos/)                        | [`boot-to-talos`][btt] का उपयोग करके Cozystack-विशिष्ट Talos Linux वितरण इंस्टॉल करें; यह संभवतः इंस्टॉल करने का सबसे आसान तरीका है। |
| 2. [Kubernetes क्लस्टर इंस्टॉल और बूटस्ट्रैप करें](/docs/v1.4/getting-started/install-kubernetes/) | Cozystack के लिए बनाए गए Talos कॉन्फ़िगरेशन प्रबंधन टूल [Talm][talm] का उपयोग करके Kubernetes क्लस्टर बूटस्ट्रैप करें।          |
| 3. [Cozystack इंस्टॉल और कॉन्फ़िगर करें](/docs/v1.4/getting-started/install-cozystack/)          | Cozystack इंस्टॉल करें, प्रशासनिक पहुँच प्राप्त करें, बुनियादी कॉन्फ़िगरेशन करें और Cozystack डैशबोर्ड तक पहुँचें।              |
| 4. [उपयोगकर्ताओं और टीमों के लिए टेनेंट बनाएँ](/docs/v1.4/getting-started/create-tenant/)         | एक उपयोगकर्ता टेनेंट बनाएँ — Cozystack में RBAC की नींव — और डैशबोर्ड तथा Cozystack API के माध्यम से उस तक पहुँचें।            |
| 5. [प्रबंधित एप्लिकेशन परिनियोजित करें](/docs/v1.4/getting-started/deploy-app/)                  | Cozystack का उपयोग शुरू करें: एक वर्चुअल मशीन, प्रबंधित एप्लिकेशन और टेनेंट के लिए Kubernetes क्लस्टर परिनियोजित करें।         |

[btt]: https://github.com/cozystack/boot-to-talos
[talm]: https://github.com/cozystack/talm
