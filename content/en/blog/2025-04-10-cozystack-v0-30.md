### Cozystack v0.30: проброс GPU, WorkloadMonitor для PVC и IP, CPUManager и автоматизированное тестирование в CI

Версия 0.30 привносит на платформу возможности GPU, а также улучшает отслеживание ресурсов и инфраструктуру тестирования.

### Основные возможности и улучшения

#### Проброс GPU для виртуальных машин

Теперь Cozystack включает **GPU Operator** и поддерживает **проброс GPU** для виртуальных машин. Пользователи могут назначать GPU виртуальным машинам прямо из панели управления, что делает возможными рабочие нагрузки AI/ML, рендеринг и другие задачи с ускорением на GPU.

#### WorkloadMonitor для PVC и публичных IP

Теперь система WorkloadMonitor отслеживает **PersistentVolumeClaims** и **использование публичных IP** по каждому арендатору. Это даёт операторам представление о потреблении ресурсов хранилища и сети на всей платформе.

#### CPUManager для выделенных инстансов

Теперь KubeVirt может запускать **типы инстансов CX и RT** с включённым CPUManager, предоставляя выделенные ядра CPU для чувствительных к производительности рабочих нагрузок.

#### Триггер переконфигурации cloud-init

Теперь пользователи могут **запускать переконфигурацию cloud-init** на работающих виртуальных машинах с помощью новой настройки конфигурации, без необходимости пересоздавать виртуальную машину.

#### VPA для VMAgent

Теперь VerticalPodAutoscaler настроен для VMAgent, благодаря чему агенты мониторинга автоматически масштабируют потребление ресурсов.

#### Автоматизированное тестирование релизов

Теперь тесты запускаются автоматически в рамках конвейера релиза, с поддержкой параллельного выполнения тестов.

- Имена GPU добавлены в спецификации виртуальных машин.
- Исправлен MetalLB с помощью патча из upstream.
- Kube-OVN обновлён до v1.13.11.
