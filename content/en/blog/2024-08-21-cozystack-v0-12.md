### Cozystack v0.12: StorageClass для всех приложений, Cilium v1.16, настройка виртуальных машин и E2E-песочница

Версия 0.12 приносит гибкость хранилища, обновлённую сеть и улучшенную инфраструктуру тестирования.

### Основные возможности и улучшения

#### StorageClass для всех приложений

Все приложения теперь поддерживают **опцию `storageClass`**, позволяющую операторам выбирать, какой класс хранилища использовать для каждого приложения. Это особенно важно для кластеров с несколькими бэкендами хранилища.

#### Cilium v1.16.1

Cilium обновлён до **v1.16.1**, что приносит улучшения производительности и новые сетевые возможности.

#### Переопределение значений для Kubernetes арендаторов

Операторы теперь могут **переопределять значения** для кластеров Kubernetes арендаторов, обеспечивая точный контроль над конфигурацией кластера.

#### Обновления в настройке виртуальных машин

Конфигурация чарта виртуальных машин и документация улучшены: заданы более удачные значения по умолчанию и понятнее описаны опции.

#### Песочница для E2E-тестирования

Новая **песочница для E2E-тестирования** упрощает контрибьюторам запуск и разработку интеграционных тестов.

#### Важное замечание об обновлении

После обновления платформы как можно скорее обновите приложения Kubernetes. В этот релиз входит исправление меток CSI-драйвера. Если CSI-драйвер перестанет работать с новыми томами, запустите скрипты миграции из `scripts/migrations/4`.
