### Cozystack v0.1: поддержка ZFS, выбор лидера и перенос документации

Первый функциональный релиз Cozystack закладывает основу для платформы производственного уровня.

### Основные возможности и улучшения

#### Поддержка ZFS

Cozystack теперь поддерживает **ZFS** в качестве бэкенда хранилища, предоставляя операторам выбор проверенного временем хранилища для их инфраструктуры.

#### Выбор лидера

Для контроллера Cozystack включён выбор лидера, что гарантирует активность только одного экземпляра в каждый момент времени — необходимое условие для развёртываний с высокой доступностью.

#### Документация

Документация проекта перенесена на [cozystack.io](https://cozystack.io), вместе с новыми руководствами Adopters, Code of Conduct, Contributing и Maintainers.

#### Прочие улучшения

- Декларативная очистка Redis для панели управления.
- Поддержка удалённых сборок Docker с `--tar-to-stdout`.
- Образ установщика переименован из `talos-installer` в `cozystack` во избежание путаницы.

Все изменения: [v0.1.0](https://github.com/cozystack/cozystack/releases/tag/v0.1.0)
