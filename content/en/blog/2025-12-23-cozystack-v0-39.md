### Cozystack v0.39: маршрутизация с учётом топологии, планирование виртуальных машин Windows, переработка Talm и VMAgent для арендаторов

Версия 0.39 улучшает работу с сетью, планирование и инструментарий для управления инфраструктурой Cozystack.

### Основные возможности и улучшения

#### Маршрутизация с учётом топологии для Cilium

Сервисы Cilium теперь поддерживают **маршрутизацию с учётом топологии**, по возможности удерживая трафик в пределах одной зоны или региона. Это снижает задержки и затраты на межзональный сетевой трафик.

#### Автоматическая раскатка подов Cilium

Поды Cilium теперь **автоматически раскатываются** при изменении их конфигурации, что избавляет от необходимости вручную перезапускать их после обновления конфигурации.

#### Планирование виртуальных машин Windows

Виртуальные машины под управлением **Windows** теперь можно планировать с помощью **nodeAffinity** через ConfigMap cozystack-scheduling, что гарантирует размещение Windows-ВМ на подходящих узлах.

#### Масштабная переработка инструмента Talm

Инструмент управления конфигурацией `talm` для Talos Linux получает значительное обновление:

- Обязательный флаг `--name` для явного указания целевого кластера.
- Сохранение YAML-документов при обработке.
- Раскрытие каталогов и автоматическое определение корня.
- Команда `kubeconfig --login` для простой аутентификации.
- **Поддержка шифрования** для секретов.
- Обёртка над `talosctl` для единообразной работы с CLI.

#### VMAgent для пространств имён арендаторов

**VMAgent** теперь развёртывается для сбора метрик из пространств имён арендаторов, обеспечивая наблюдаемость мониторинга по каждому арендатору.

### Исправления (v0.39.1–v0.39.5)

- Фильтр по уровню серьёзности в Slack для Alerta.
- Виртуальные машины всегда публикуются как минимум через сервис ClusterIP для внутрикластерного DNS.
- Исходящий трафик арендатора разрешён к родительским ingress-подам.
- SeaweedFS обновлён до v4.05 с локальностью трафика и панелями управления Grafana.
- Kube-OVN обновлён до v1.14.25.
- Критические патчи piraeus-server для стабильности DRBD.
- Исправлено состояние гонки DRBD в updateDiscGran.
