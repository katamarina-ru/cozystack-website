Это руководство проведёт вас через первое развёртывание кластера Cozystack.
По ходу работы вы познакомитесь с ключевыми концепциями, научитесь работать с Cozystack через панель управления и CLI
и получите рабочий PoC.

Руководство разбито на несколько шагов.
Обязательно завершайте каждый шаг перед началом следующего:

| Шаг                                                                               | Описание                                                                                                                       |
|-----------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| [Требования: подготовка инфраструктуры и инструментов](/docs/v1.4/getting-started/requirements/) | Подготовьте инфраструктуру и установите необходимые CLI-инструменты на локальную машину перед выполнением этого руководства.   |
| 1. [Установка Talos Linux](/docs/v1.4/getting-started/install-talos/)                           | Установите дистрибутив Talos Linux, адаптированный для Cozystack, с помощью [`boot-to-talos`][btt]; это, вероятно, самый простой способ установки. |
| 2. [Установка и инициализация кластера Kubernetes](/docs/v1.4/getting-started/install-kubernetes/) | Инициализируйте кластер Kubernetes с помощью [Talm][talm], инструмента управления конфигурацией Talos, созданного для Cozystack. |
| 3. [Установка и настройка Cozystack](/docs/v1.4/getting-started/install-cozystack/)             | Установите Cozystack, получите административный доступ, выполните базовую настройку и откройте панель управления Cozystack.    |
| 4. [Создание тенанта для пользователей и команд](/docs/v1.4/getting-started/create-tenant/)     | Создайте пользовательский тенант — основу RBAC в Cozystack — и получите доступ к нему через панель управления и Cozystack API. |
| 5. [Развёртывание управляемых приложений](/docs/v1.4/getting-started/deploy-app/)               | Начните пользоваться Cozystack: разверните виртуальную машину, управляемое приложение и кластер Kubernetes для тенанта.        |

[btt]: https://github.com/cozystack/boot-to-talos
[talm]: https://github.com/cozystack/talm
