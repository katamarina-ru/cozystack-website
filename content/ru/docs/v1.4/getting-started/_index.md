Это руководство проведёт вас через первое развёртывание кластера Cozystack.
По пути вы познакомитесь с ключевыми концепциями, научитесь работать с Cozystack через дашборд и CLI
и получите рабочий proof-of-concept.

Руководство разбито на несколько шагов.
Обязательно завершайте каждый шаг перед началом следующего:

| Шаг                                                                               | Описание                                                                                                                       |
|-----------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| [Требования: подготовка инфраструктуры и инструментов]({{% ref "requirements" %}}) | Подготовьте инфраструктуру и установите необходимые CLI-инструменты на свою машину перед началом руководства.                  |
| 1. [Установка Talos Linux]({{% ref "install-talos" %}})                           | Установите специальную для Cozystack сборку Talos Linux с помощью [`boot-to-talos`][btt] — вероятно, самый простой способ.      |
| 2. [Установка и инициализация кластера Kubernetes]({{% ref "install-kubernetes" %}}) | Инициализируйте кластер Kubernetes с помощью [Talm][talm], инструмента управления конфигурацией Talos, созданного для Cozystack. |
| 3. [Установка и настройка Cozystack]({{% ref "install-cozystack" %}})             | Установите Cozystack, получите административный доступ, выполните базовую настройку и откройте дашборд Cozystack.                |
| 4. [Создание тенанта для пользователей и команд]({{% ref "create-tenant" %}})     | Создайте пользовательский тенант — основу RBAC в Cozystack — и получите к нему доступ через дашборд и Cozystack API.            |
| 5. [Развёртывание управляемых приложений]({{% ref "deploy-app" %}})               | Начните пользоваться Cozystack: разверните виртуальную машину, управляемое приложение и тенант-кластер Kubernetes.             |

[btt]: https://github.com/cozystack/boot-to-talos
[talm]: https://github.com/cozystack/talm
